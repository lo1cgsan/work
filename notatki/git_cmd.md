# Git – polecenia

1) Połączenie repozytorium lokalnego ze zdalnym:

    git remote add origin <remote_repo_URL>
    git push --all origin

2) Synchronizacja lokalnej gałęzi z repozytorium zdalnym:

    git push <remote-name> <branch-name>
