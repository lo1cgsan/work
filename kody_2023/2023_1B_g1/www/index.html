Podstawy budowy stron WWW

Technologie WWW:

  1. HTML (HyperText Markup Language) – hipertekstowy język znaczników określający strukturę strony
  2. CSS (Cascading Style Sheets) – kaskadowe arkusze stylów określające wygląd strony
  3. JavaScript – skryptowy język programowania obsługujący zdarzenia na stronie WWW

Struktura strony WWW

  • Nagłówek strony – <head>
  • Ciało strony – <body>
